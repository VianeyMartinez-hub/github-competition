# GitHub Competition

Maze-style GitHub competition for beginners to practice branches, commits, merges, and pull requests.

## Welcome

Your journey through the temple begins now.

Each branch in this repository represents a checkpoint. At every checkpoint, you will complete tasks, solve puzzles, and collect clues that lead you to the final treasure.

Your competition starts at: `temple-entrance`

---

## How to Use This Repository

Follow these steps as you progress:

1. Use the terminal to switch to the correct branch by running:
   git checkout <branch-name>

2. In each branch, open and read the README.md carefully.

3. Follow all instructions in the README.

4. Open any additional files mentioned (such as .txt files) to find clues and complete the task.

5. Once you finish the task, move to the next branch as instructed.

---

## Important Notes

- Complete all checkpoints in order. Do not skip steps.
- Keep track of every clue word your team finds.
- Make sure all team members take turns using the keyboard.
- If you feel stuck, re-read the README and check all files in the branch.

---

Good luck. The treasure awaits.
