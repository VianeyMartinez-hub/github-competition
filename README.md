# Hidden Treasure

You made it.

The temple only opens for teams who collected every clue and worked together.

## Final Task

1. Open `final.txt`
2. Solve the puzzle to find the final word
3. Combine all your clue words (order matters)
4. Create a file named `treasure.txt`
5. Write the final phrase
6. Push your work
7. Open a pull request

Your pull request title must start with:
`Team: <your team name>`

Example:
`Team: Temple Raiders`

Only the correct phrase will unlock the treasure. Make sure it's in all CAPS!
